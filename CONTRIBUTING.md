# Contributing

Thank you for your interest in Chess Game — it genuinely means a lot.

This project is a **personal learning project** and is intentionally kept as a solo endeavour. I want to maintain full ownership of the code and its direction, so I am not accepting pull requests, feature suggestions, or code contributions of any kind at this time.

### What you *can* do

- **Report bugs** by opening an issue — this is always helpful and appreciated
- **Fork the project** if you'd like to build on it for your own learning
- **Star the repo** if you enjoy the project

### What I ask you not to do

- Open pull requests — they will be closed without review
- Submit unsolicited feature requests — the roadmap is driven entirely by my own goals

I appreciate your understanding and I'm glad the project caught your attention.
